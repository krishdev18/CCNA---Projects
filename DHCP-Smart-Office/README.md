# 🏢 Smart Office Network – DHCP Implementation

## 📌 Project Overview

This project demonstrates the implementation of Dynamic Host Configuration Protocol (DHCP) in a small office network using Cisco Packet Tracer.

The goal is to automate IP address assignment and reduce manual configuration errors.

---

## 🧠 Scenario

A company expanded into two departments:

* Development Team
* Testing Team

Manual IP assignment caused network issues. A centralized DHCP solution was implemented using a router.

---

## 🛠 Technologies Used

* Cisco Packet Tracer
* DHCP
* Basic Routing
* Switching

---

## 🌐 Network Design

* 1 Router (DHCP Server)
* 2 Switches
* Multiple PCs

---

## ⚙️ Features Implemented

* Automatic IP allocation
* DHCP pools configuration
* Default gateway assignment
* DNS setup
