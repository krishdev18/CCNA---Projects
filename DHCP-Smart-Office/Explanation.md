# DHCP Project Explanation

## What is DHCP?
DHCP automatically assigns IP addresses to devices in a network.

## Why DHCP?
- Reduces manual work
- Avoids IP conflicts
- Scalable

## How it Works
1. Client sends DHCP Discover
2. Server replies with Offer
3. Client sends Request
4. Server sends Acknowledgment

## Result
All PCs received IP automatically from router.
